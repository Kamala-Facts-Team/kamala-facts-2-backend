## Installation

Clone this repo:

https://github.com/Kamala-Facts-Team/kamala_facts_2_backend

Install Dependicies:

    $ bundle install

Create Database:

    $ rails db:create 

Migrate Database:

    $ rails db:migrate
    
Seed Database with sample data:

    $ rails db:seed    

Start Server:

    $ rails s
